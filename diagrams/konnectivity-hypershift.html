Konnectivity in HyperShift
How the control plane tunnels traffic to worker nodes when they live in separate networks.
⚠ HyperShift / Hosted Control Planes. In HyperShift the control plane runs in a management cluster (different network) from the worker nodes. Konnectivity provides a reverse tunnel so the API server, operators, and proxy sidecars can reach kubelets, services, and endpoints on the data plane.
