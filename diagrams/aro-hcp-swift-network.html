ARO HCP Swift Network Architecture
How Swift (VNet Integration) enables fully private control plane access to customer Key Vault.
⚠ ARO HCP with Swift. Swift allows the ARO HCP control plane to inject a NIC directly into a delegated subnet in the customer's VNet. This eliminates public internet exposure for Key Vault access — all etcd encryption key operations stay on Azure's private backbone.
